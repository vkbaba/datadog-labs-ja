このワークショップでは、EC サイト「Swagstore」を題材に、Datadog で SLO を設定・監視し、問題発生時に自動的に対処するワークフローを構築する方法を学びます。

## 学習内容

- Datadog によるアプリケーションデータの収集確認
- ログ・トレースを使った問題の特定
- APM メトリクスをもとにした SLO の作成
- バーンレートモニターの設定
- サービスマップによる依存関係の把握
- Workflow Automation による自動修復の実装
